Document Q&A With RAG

A grounded document question-answering system that retrieves PDF chunks, scores relevance, cites source pages, and refuses weak-context answers instead of hallucinating.

Chunk quality scoring shows exactly which evidence was retrieved. Source attribution returns document name, page, preview, and confidence. Failure gate returns “I don’t have enough information” for weak matches.

#2 · Structured Outputs

Structured Data Extraction

A typed extraction system that converts job postings into validated Pydantic objects with salary normalization, batch processing, and field-level accuracy metrics.

Pydantic catches impossible salary ranges and wrong field types. Batch processor is designed for 100+ postings with async execution. Metrics report precision, recall, exact match, and per-field scores.

Two-Tower Retrieval System

A scalable e-commerce search system with separate query and product towers, metadata boosts, personalization controls, and ranking evaluation.

Query tower and item tower embed search intent and product catalog features. Metadata signals include category, price, rating, review count, and popularity. Personalization changes rankings by segment, preferred category, and price sensitivity.
